# Trabajo Final – Ciencia de Datos (UIMP)

## 📊 Análisis y Modelado de Datos

Trabajo final de la asignatura **Ciencia de Datos** de la **Universidad Internacional Menéndez Pelayo (UIMP)**. El objetivo del proyecto es aplicar una metodología completa de Ciencia de Datos sobre un conjunto de datos real, desde la exploración inicial hasta la evaluación de modelos predictivos.

---

## 🧠 Objetivos

- Aplicar una metodología completa de Ciencia de Datos (CRISP-DM / KDD).
- Realizar análisis exploratorio de datos (EDA).
- Preprocesar y transformar los datos.
- Construir y evaluar modelos de aprendizaje automático.
- Interpretar resultados y extraer conclusiones.

---

---

## 🔍 Metodología

1. Comprensión del problema
2. Exploración de los datos
3. Preprocesamiento
4. Modelado
5. Evaluación
6. Conclusiones

---

## 🛠️ Tecnologías

Entre otras:

- Python 3
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
- Jupyter Notebook
